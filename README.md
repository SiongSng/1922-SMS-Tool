# 1922 實聯制掃描器

![logo](https://github.com/SiongSng/1922-SMS-Tool/blob/main/assets/logo.png?raw=true)
## 介紹
台灣 1922 實聯制掃描器，支援 Android 與 IOS 手機。 

## 特色
- 支援開關手電筒，再暗的位置都能掃描！
- 支援從背景直接發送訊息，免去手動按下發送按鈕的步驟 (此功能僅支援 Android)
- 支援查看簡訊實聯制發送紀錄
- 開放原始碼的自由軟體、無任何廣告
